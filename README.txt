The aim of project is to create a food sharing web application based on the Express (Node.js) framework, MongoDB database. This structure is well suited for the crud (create, read, update and delete) application design functionalities. 

How to run it?

1.please download Node.js and MongoDB

2.run MongoDB

3.command line 

npm install

node app.js

open browser : http://127.0.0.1:3000
